# 字尾
1. *-mab*：單株抗體，作用在細胞外受體，幾乎都**注射使用**
	- *-omab* 全鼠源 (murine)，最容易過敏
	- *-ximab* 嵌合型 (ch**i**meric)
	- *-zumab* 人化型 (humani**z**ed)
	- *-umab* 全人型 (fully human)
2. TKI 全都是 *-nib*：小分子藥物，**nibble**(蠶食) 進細胞裡，屬於小分子干擾素（像吃掉訊號）
	- *-brutinib* = BTK 類，可以治 B 細胞相關的血癌
3. Ser/Thr I
	1. *-ciclib* = Cyclin-Dependent Kinase，圓圓的 → 乳癌
	2. 轉移性黑色素乳癌
		- *rapid + 雙重特異性抑制劑*，BRAF → MEK 是 下游訊號通路，兩者常合併使用
		- *-rafenib / raf* = B-**rap**idly accelerated fibrosarcoma
		- *-metinib / me* = MAPK/ERK Kinase 雙重特異性抑制劑 - **並用減少抗藥性**
	3. *-olimus* = mTOR，除了 Tacrolimus 是作用在 Calcineurin，該類常與免疫調節有關
	4. *-lisib* = Phosphatidy**linositol** 3-Kinase，脂質激酶，三條線(li i) → 治**淋**巴瘤
	5. *-par* = PARP（Poly-ADP ribose polymerase）
4. 免疫檢查點抑制劑
	1. *-limumab* = T-cell 上 CTLA-4
