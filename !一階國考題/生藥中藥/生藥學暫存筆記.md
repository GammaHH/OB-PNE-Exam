# 配醣體



##### 試劑、試驗
- KK 試驗(Keller Kiliani test) - 紅棕色變成藍色 (不重要)
- 斐林試劑 - 測還原醣、多醣類（Polysaccharides）
# 生物鹼

- 雜環不含 N(非典型) = 摩卡沒揪
- Isoquinoline 結構整理
- 鴉片整理
- berberine 口訣，連博士
- tropane 生物鹼成員，以及非茄科的
- tropane 合成途徑
- **各項生物鹼成員整理**
- 藜蘆比較 
- Indole 類生物鹼環整理
##### 試劑、試驗
- Thalleioquin test：金雞納皮測驗，加入胺及溴試劑，有含呈**翠綠色**
