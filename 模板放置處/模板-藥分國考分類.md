# 對比與應用

|**分類**|**傳統分析法**|**儀器分析法**|
|---|---|---|
|**應用範圍**|適用於單純成分檢測|複雜成分分離與定性定量|
|**靈敏度**|一般靈敏度|高靈敏度|
|**儀器需求**|簡單的實驗儀器|需要專業儀器如UV、HPLC、NMR等|
|**典型範例**|酸鹼滴定、比色分析、重量分析|高效液相層析、氣相層析、質譜、原子吸收光譜法|

上述方法在藥師國考中的考試比重不等，其中儀器分析多用於現代藥物的質量控制，而傳統分析法仍在基礎檢驗中占有一席之地。
